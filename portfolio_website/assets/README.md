# Assets Folder

Place your AI-generated background image here.

## Background Image Requirements

- **Filename**: `background-image.jpg`
- **Recommended Size**: 1920x1080px or higher
- **Format**: JPG or PNG
- **Style**: Should be subtle and minimalist, blending seamlessly with the glassmorphism theme

The image will be used on the landing page with a blur effect to create an integrated, seamless background.

If no image is provided, the site will automatically use a gradient fallback.
